\# 🔧 P2P Classroom Communication System



This project is a \*\*real Peer‑to‑Peer (P2P)\*\* communication system that simulates up to \*\*19 computers\*\* (1 teacher + 18 students) and enables real‑time messaging and file sharing.



There is \*\*no central server\*\* – all data flows \*\*directly between peers\*\* using PeerJS.



\---



\## 💬 Chat \& File Sharing



\- Type a message into the input field → it is sent to the \*\*first connected peer\*\*  

\- Send using \*\*Enter\*\* or the \*“Send Message”\* button  

\- \*\*Group Message\*\* sends the message to \*all\* connected peers  

\- Files are also sent to all connected peers  

\- Each message displays:

&#x20; - sender

&#x20; - timestamp



\---



\## 📁 File Transfer



\- Send images, PDFs, documents  

\- Images show a live preview inside the chat  

\- Other files can be downloaded



\---



\## 📊 Visual Feedback



\- Connected peers are highlighted with \*\*color changes\*\*  

\- Real‑time status indicators  

\- Continuously updated connection list



\---



\## 💡 Multi‑Device Testing



\- Open the app in multiple browser windows or on different devices  

\- Share Peer IDs to connect  

\- Each device receives a predefined ID  

&#x20; - e.g., `classroom-teacher`, `classroom-student1`, `classroom-student2`, …



\---



\## 🔍 Automatic Peer Discovery



\- On initialization, the system automatically scans for \*\*online peers\*\*  

\- Only online peers are displayed  

\- You can connect only to online peers



\---



\## 🖱️ Easy Connection



\- Click any online peer → automatic connection  

\- You can connect to multiple peers simultaneously  

\- The list of connected peers updates in real time



\---



\## 📌 Feature Summary



\- Real P2P communication using PeerJS  

\- 19‑device classroom simulation  

\- Chat, group messages, file sharing  

\- Automatic peer discovery  

\- Real‑time status and visual feedback  

\- Simple click‑to‑connect interface
